# README

### Synopsis

A VS2019 C++ wxWidgets desktop application to change Windows backgrounds.

### Usage

The application takes the canonical path of a directory with JPEG image files 
as its sole command line argument.

### Structure

**Directories**

- *bin*
  - Base directory of all binary files
- *doc*
  - Base directory for project documentation
- *etc*
  - Directory for all configuration files
- *log*
  - Log file directory
- *src*
  - Base directory for all source code
- *var*
  - Base directory for temporary files

**Files**

- *BackgroundChanger.rc*
  - This project's Windows resource script
- *BackgroundChanger.sln*
  - Project solution file
- *BackgroundChanger.vcxproj*
  - Project VS2019 definition file
- *BackgroundChanger.vcxproj.filters*
  - Solutions explorer filters file
- *backgroundchanger_icon_256x256.ico*
  - Application icon file
- *resource.h*
  - Numeric ID definitions file for the Windows resource script
